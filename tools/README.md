# 工具目录说明

## 工具信息 JSON 文件

根目录可以包含一个 `tool.json` 或 `info.json` 文件来提供工具的详细信息。

### JSON 文件格式

```json
{
  "name": "工具显示名称",
  "description": "工具的详细描述信息",
  "version": "版本号（可选）",
  "author": "作者名称（可选）"
}
```

### 字段说明

- **name** (可选): 工具的显示名称。如果未提供，将使用目录名称。
- **description** (可选): 工具的详细描述。如果未提供，将根据目录名称自动生成。
- **version** (可选): 工具版本号。
- **author** (可选): 工具作者名称。

### 注意事项

- JSON 文件必须使用 UTF-8 编码
- 如果工具目录中没有 JSON 文件，系统将使用默认逻辑：
  - 名称：使用目录名称
  - 描述：根据目录名称自动生成
- 支持的文件名：`tool.json` 或 `info.json`（优先使用 `tool.json`）
